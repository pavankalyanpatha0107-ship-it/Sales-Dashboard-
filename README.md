# 📊 Sales Analysis Dashboard

## 📌 Project Overview

This project showcases an interactive **Sales Analysis Dashboard** built using **Power BI**. It provides key insights into sales performance, profit, quantity, and customer behavior across different regions, categories, and time periods.

The dashboard is designed for **data-driven decision-making**, helping businesses understand trends and improve performance.

---

## 🚀 Key Features

* 📍 **Region-wise Analysis** (Central, South, East, West)

* 📅 **Year-wise Filtering** (2014–2017)

* 💰 **KPIs Overview**

  * Total Sales: 484.2K
  * Total Profit: 49.5K
  * Quantity Sold: 7581
  * Discount: 315.5

* 📈 **Sales Trends**

  * Monthly sales analysis
  * Quarterly performance breakdown

* 📊 **Category Insights**

  * Technology, Furniture, Office Supplies comparison
  * Category-wise profit distribution

* 🌍 **Geographical Analysis**

  * Sales comparison by country

* 🚚 **Shipping Mode Analysis**

  * Standard Class, Second Class, First Class

* 👥 **Customer Segment Analysis**

  * Consumer, Corporate, Home Office

* 🏆 **Top 10 Products**

  * Best-performing products based on sales

## ▶️ How to Use

1. Download the `.pbix` file from this repository
2. Open it using **Power BI Desktop**
3. Explore filters:

   * Region
   * Year
4. Interact with visuals to gain insights

---

## 💡 Insights Gained

* Technology category generates the highest sales
* Standard Class shipping is most commonly used
* Consumer segment contributes the largest share
* Sales show strong growth in later months
